
## What this does?
This repo along with https://github.com/salemgaleb3-devops/k8smanifest-gitops.git creates a Jenkins pipeline with GitOps to deploy code into a Kubernetes cluster. CI part is done via Jenkins and CD part via ArgoCD (GitOps).


## Jenkins installation
to install Jenkins go to https://github.com/salemgaleb3-devops/jenkins-with-docker-engine.git

## ArgoCD installation

Install ArgoCD in your Kubernetes cluster following this link https://argo-cd.readthedocs.io/en/stable/getting_started/
